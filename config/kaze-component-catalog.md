# Kaze Component Catalog

## Purpose
This file lists confirmed Kaze UI components available for implementation.

AI and coding agents must not invent Kaze component names.

If a component is not listed here, mark it as:

Unknown / verify from Kaze

## General Rules
- Use Kaze components where available.
- Do not use raw HTML controls if a Kaze equivalent exists.
- Do not use Ant Design directly if Kaze wraps it.
- Use Font Awesome icons unless custom assets are provided.
- Use standard Kaze loading, empty, error, disabled, validation, and modal states unless custom states are shown.

## Confirmed Kaze Components

| UI Pattern | Confirmed Kaze Component | Notes |
|---|---|---|
| Button | KazeButton | Use for normal, icon, and action buttons |
| Text Input | KazeInput | Use for text input |
| Select / Dropdown | KazeSelect | Use for option selection |
| Table / Listing | KazeDataTable | Use for listing pages |
| Modal / Dialog | KazeModal | Use for confirmation/dialog flows |
| Badge / Status | KazeBadge | Use for status labels |
| Alert / Message | KazeAlert | Use for warning, error, success messages |
| Form | KazeForm | Use for form structure |
| Form Field | KazeFormItem | Use for validation field wrapper |
| Date Picker | KazeDatePicker | Use for date input |
| Tabs | KazeTabs | Use for tab navigation |
